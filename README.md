# ALU Multiplication Microprogram

### Описание
Разработка микропрограммы умножения целых чисел с фиксированной запятой на основе ALU-1. Программа выполняет умножение 8-разрядных чисел и записывает 16-разрядный результат в регистры C и D.

### Основные этапы:
1. Инициализация регистров (Ax, Bx, C, D).
2. Проверка младшего бита сомножителя.
3. Сдвиги регистров и сложение.
4. Запись результата в регистры C и D.

### Тестирование:
- 177 × 77: результат 13629.
- 3 × 4: результат 12.
- 0 × 10: результат 0.
